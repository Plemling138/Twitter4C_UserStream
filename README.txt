１．はじめに
・TwitterのユーザストリームをTwitter4Cで扱うときのサンプルコードです。
　ビルドすると、ツイートと自分宛てのイベント（の一部）を見ることができます
・あくまでサンプルなので、無駄が多いです
・終了はCtrl+Cです（笑）

２．動作環境
・gccとOpenSSLが使用できる環境であれば基本的に実行可能だと思われます
  こちらではCygwin(x64)、CentOS 6.5 x86_64で動作を確認しています
・文字エンコーディングはUTF-8を使用してください
